# AutoOpomena — demo

Prijedlog rješenja za automatsku pripremu opomena iz kartica kupaca.

- `opis.html` — opis rješenja + pitanja za dogovor
- `dashboard.html` — klikabilni demo dashboard (izmišljeni podaci, bez backenda)
- `prezentacija.html` — prezentacija proizvoda

Samostojeći HTML (Tailwind CDN + Alpine.js) — potreban internet zbog CDN-a.
